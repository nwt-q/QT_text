# 记事本

产品基于Window中的记事本进行开发并实现其基础功能, 项目功能包括,打开,关闭,保存文件; 项目包含优美的外观,以及简单的操作性,对于用户体验考虑到第一, 后续更甚会增加markdown格式等等

#### 目录结构说明

documents -- 项目开发相关文档（如：项目需求文档、设计文档等等）

QT_text -- 项目源码（比如：前端项目、后端项目等等）
